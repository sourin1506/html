Ordered List

An ordered list created using the < ol> element and each list item starts with the < li> element. Ordered lists are used when the order of the list's items is important. The list items in an ordered list are marked with numbers or letters. The type attribute of the < ol> tag, defines the type of the list item marker. Here are a few examples:

Example 1:

<html>
<head>
</head>
<h2>An ordered HTML list with number</h2>
<ol type="1">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>
</body>
</html>

This will produce the following result:

  1. Coffee
  2. Tea
  3. Milk

<html>
<head>
</head>
